# Gleano website

Static marketing site for [Gleano](https://github.com/Vanduea/gleano): landing page, demo (`demo.html`), and privacy copy.

**Live:** enable [GitHub Pages](https://docs.github.com/en/pages) on this repository (Settings → Pages → deploy from branch, usually `main` / root or `/docs`).

APK files are **not** stored in git (see `downloads/README.md`). Attach builds under **Releases → Assets**.
